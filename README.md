# File Organizer 🐟

Automatically sorts and organizes your files 
into folders using Python.

## What it does
- 📸 Images → /Images
- 📄 Documents → /Documents
- 🎵 Music → /Music
- 🎬 Videos → /Videos
- 📦 Archives → /Archives

## How to use
1. Run the script
2. Enter your folder path
3. Done! Your files are organized!

## Built with
- Python
- os library
- shutil library
